absolute + margin

多行文字的垂直居中技巧，但居中的元素具有固定尺寸。适合做图片等居中效果。

absolute + margin auto

缺点也是定位元素必须有固定的宽高。

absolute + translate

先利用定位，左、上各位移50%，然后自身再向相反的方向位移50%。

flex + align-items

利用display: flex;布局。代码干净整洁。老一版浏览器兼容性问题。

flex + :before + flex-grow

flex垂直方向布局。添加一个伪元素，该伪元素利用flex-grow: .5;取到剩余空间的一半，然后把需要居中的元素推到中间位置。

flex + margin auto

父元素设置flex，然后子元素设置margin auto。IE中会存在兼容性问题。

flex + align-self

父元素设置flex，利用子元素可以自己设定自己的垂直居中方式。

display: table-cell

缺点是宽度要设置实际尺寸，百分比没效果。 较老的居中方式，原理是利用css的display属性将div设置成表格的单元格，这样就能利用支持存储单元格对齐的vertical-align属性来将信息垂直居中了。

calc

这个方法不能大量使用，否则会降低网页性能。
缺点就是居中的元素需要知道具体尺寸。

relative + translateY

原理和relative + absolute相似，只是这里是相对自己位移。

padding

一种简单的居中方式。严格来讲，只是设置上下内边距大小一样。
父容器的高度是弹性的。
